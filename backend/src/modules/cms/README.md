# CMS

This module is responsible for storing and loading content displayed on Roobet.com

## Working with Markdown

Rewriting documents as Markdown can be cumbersome, the following tools can help with that. Make sure
to review the output before saving in the ACP.

Conversion tools:

- [Word to Markdown](https://word2md.com/)
- [Paste to Markdown](https://euangoddard.github.io/clipboard2markdown/)

Preview tools:

- [Markdown Live Preview](https://markdownlivepreview.com/)
