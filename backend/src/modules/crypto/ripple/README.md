# Ripple

## Testing XRP

This is only for the XRP testnet.

- Generate credentials here: https://xrpl.org/xrp-testnet-faucet.html
  - Select testnet and click the button to generate creds
  - the new account should have roughly 10k testXRP
- Use the wallet UI here: https://test.xrptoolkit.com/connect-wallet
  - Select the secret key option and enter in your newly generated creds
  - Now you can send XRP and manage your testnet Account
