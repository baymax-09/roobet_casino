# King of the Hill

This module provides the backing for our gamification system: King of the Hill.
There are two types of KOTH events: Astro Roo and King Roo.

## Types of KOTH Events

### Astro Roo

The current Astro Roo takes a share from every wager on Crash.
To become Astro Roo, a user must ride a crash further than the current Astro Roo, with a minimum wager of $5.

### King Roo

The current King Roo takes a share from every wager across all slots.
To become King Roo, a user needs to hit a 100x multiplier on any slot, with a minimum wager of $1.
