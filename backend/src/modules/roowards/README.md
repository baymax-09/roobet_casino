# Roowards

Roowards is a gamification feature that rewards recurring actions on the site. There are three types of Roowards: daily, weekly, and monthly. You can separately level up each of these types of Roowards to get a higher percentage of each Rooward. Roowards must be claimed and are not automatically awarded.

## Types of Roowards

### Daily

Daily rakeback, based on your wagers yesterday.

### Weekly

Weekly cash bonus, based on lossback percentage from the week prior.

### Monthly

Monthly cash bonus, based on lossback percentage from the month prior.
